```bash
# Expected: {"status":"ok","uptime_seconds":...,"version":"1.0.0"}
```

**Actual result:**
```json

  "status": "ok",
  "uptime_seconds": 109.5,

  "timestamp": "2026-04-17T11:31:30.919060+00:00"

```

### Readiness Check
```bash
curl https://ai-agent-ujf7.onrender.com/ready
# Expected: {"status":"ready"} or {"status":"ok"}
```

### Authentication Required (no key → 401)
```bash
curl -X POST https://ai-agent-ujf7.onrender.com/ask \
  -H "Content-Type: application/json" \
  -d '{"question": "Hello", "user_id": "user1"}'
# Expected: 401 Unauthorized
```

**Actual result:** `401 Unauthorized` ✅

### API Test (with valid key → 200)
```bash
curl -X POST https://ai-agent-ujf7.onrender.com/ask \
  -H "X-API-Key: <AGENT_API_KEY>" \
  -H "Content-Type: application/json" \
  -d '{"question": "Hello", "user_id": "user1"}'
```

**Actual result:**
```json
{
  "question": "Hello",
  "answer": "[Mock] Answer to: Hello",
  "user_id": "user1"
}
```
